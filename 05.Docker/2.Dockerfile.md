# Sample Dockerfile for nodejs application

```bash
# Use an official Node.js image as the base
FROM node:18-alpine

# Set working directory inside the container
WORKDIR /app

# Copy only package files first (for caching npm install)
COPY package*.json ./

# Install dependencies. Run commands while building container
RUN npm install --production

# Copy the rest of your source code
COPY . .

# Expose the port your app runs on
EXPOSE 3000

# Defines the command that runs when the container starts.
CMD ["node", "server.js"]
```


# How to build & run it

```bash
# Build the image
docker build -t my-node-app .

# Run the container
docker run -p 3000:3000 my-node-app
```
